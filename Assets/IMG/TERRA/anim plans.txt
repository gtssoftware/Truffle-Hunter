Terracottal is one of the more complex animations, it normally uses TLEFT, but since both arms are different, we cant just flip sprites, so when facing right it uses TRIGHT, when attacking for left it uses (for a 0.6 second animation), TLATK for the first half (0.3s), and for the second half TLATK1 (0.3s), when dead it moves down forever till it touches ground, once it touches ground it uses TDEAD. this enemy is a floating enemy.
